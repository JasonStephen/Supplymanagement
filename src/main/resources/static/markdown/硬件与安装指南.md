## 📦 如何安装
首先你需要去GitHub克隆我们的项目（在我们公开或得到许可后），在Navicat创建数据库并运行查询。其次配置好项目，包括AI API Key，数据库的配置信息，项目图片存放目录，配置Grafana图表等。安装好必要依赖后即可开始使用（Spring默认端口8081，Grafana默认端口3000）。
