# 供应链管理系统的技术文档



## 一、网页拓扑结构

网页结构遵循三部分构成，分别是顶部导航栏，内容板块和底部栏。

1. 顶部导航栏用于网站的导航，共分为 返回首页(logo)/物料中心/产品作坊/买卖中心/关于我们
2. 内容板块在每个不同栏目中略有不同，会单独介绍。
3. 底部栏同样显示菜单导航，仅在高分辨率下才会显示，低分辨率下仅显示版权信息



### 1.导航栏

#### 1.1 导航栏功能

**左上角：**用于返回主菜单的LOGO

**右上角：**物料中心，产品作坊，买卖中心，关于我们



### 2. 主页

#### 2.1 主页用途

主页通常用于显示基本的一些内容。

| 内容介绍                        | 跳转页面                | 未登录/普通用户 | 企业用户 | 管理员 | 创世神 |
| ------------------------------- | ----------------------- | --------------- | -------- | ------ | ------ |
| 用户信息                        | 5 角色 - 个人信息页     | ℹ️显示需要先登录 | ✅        | ✅      | ✅      |
| 最新产品(Top 5)                 | 1 物料中心 - 物料中心   | ✅               | ✅        | ✅      | ✅      |
| 库存告警信息(Top 5)             | 1 物料中心 - 库存管理页 | ❌               | ✅        | ✅      | ✅      |
| 销售订单(Top 5)&采购订单(Top 5) | 3 买卖中心 -            | ℹ️只显示销售订单 | ✅        | ✅      | ✅      |
| 买卖订单状态 (Top 5)            | 3 买卖中心              | ❌               | ✅        | ✅      | ✅      |

#### 2.2主页内容注意事项

##### 2.2.1 用户信息

用户信息显示：用户头像（左上），用户名“欢迎{user_name}”，你的角色是{role}

##### 2.2.2 最新产品

根据产品ID显示最新的5个产品

##### 2.2.3 库存告警消息

将库存中随机5个存在告警的产品展示出来

##### 2.2.4 销售订单和采购订单

根据order_id获取最新的5个订单（采购订单不提供给普通用户）

##### 2.2.5 买卖订单

将最新的5个未完成的订单展示给用户



### 3. 物料中心  

#### 3.1 页面用途

主页包含以下内容

| 内容介绍               | 跳转页面                      | 未登录/普通用户 | 企业用户 | 管理员 | 创世神 |
| ---------------------- | ----------------------------- | --------------- | -------- | ------ | ------ |
| 产品和材料展示         | 当前页面                      | ✅               | ✅        | ✅      | ✅      |
| 产品详情查看           | 产品详情页（点击产品div进入） | ✅               | ✅        | ✅      | ✅      |
| 产品和材料的添加和删除 | 当前页面（顶部 [管理员]）     | ❌               | ❌        | ✅      | ✅      |
| 产品和材料的编辑       | 产品详情页（顶部 [管理员]）   | ❌               | ❌        | ✅      | ✅      |
| 产品类别管理           | 产品详情页（顶部）            | ❌               | ✅        | ✅      | ✅      |
| 产品配方管理           | 产品详情页（顶部）            | ❌               | ✅        | ✅      | ✅      |
| 产品库存查看           | 产品库存页                    | ❌               | ✅        | ✅      | ✅      |
| 增加减少产品库存       | 产品库存页（[创世神]）        | ❌               | ❌        | ❌      | ✅      |

#### 3.2 物料中心页

**页面等级：一级**

**地址： /product/list**

物料中心支持：根据类别筛选，搜索产品名称

物料中心页展示产品和材料（为普通用户屏蔽是材料的产品），可以分页，内容包括如下：

| 行数             | 具体内容                         |
| ---------------- | -------------------------------- |
| 第一行           | 产品的图片（logo代加）           |
| 第二行           | 产品名称                         |
| 第三行           | 产品描述                         |
| 第四行           | 产品类别                         |
| 第五行           | 价格                             |
| 第六行【管理员】 | 产品或材料的删除按钮（需要确认） |

**注意：管理员额外能看到产品或材料的添加按钮和删除按钮**

#### 3.3 产品详情页

**页面等级：二级**

**地址：/product/details**

产品详情页展示当前产品的一些详情，具体内容包含如下。

1. **普通用户**可看到的：产品LOGO，产品名称，产品描述，产品类别，产品价格。
2. **企业用户**可看到的：产品库存，产品库存状态，产品配方信息，[产品编辑]，[产品类别管理]。

#### 3.4 库存管理页

**页面等级：二级**

**地址：/product/inventory**

库存管理页可以展示所有的产品的库存信息，以及库存告警信息。

1. **企业用户**可以进行库存告警值的编辑。
2. **创世神**可以进行库存增加或减少的操作。

### 4. 产品作坊

#### 4.1 页面用途

**页面等级：一级**

**地址：/produce**

产品作坊用户加工产品，具体可参考produce，拥有配方的产品可以被加工并立刻进行产品的生产

### 5. 买卖中心

**页面等级：一级**

**地址：/trade**

#### 5.1 页面用途

订单可以通过产品筛选，可以对采购还是销售的订单进行筛选，以及订单的监控和签收。

展示所有未销售的订单，或者未被采购的订单。具体参考相应的页面。

页面包括：供应订单管理，销售订单管理和进行中的订单的管理。（使用现成的已经完成的网页改良）

### 6. 用户中心

#### 6.1 页面用途

用户中心主要为用户提供用户信息查看的渠道，但是不同的用户拥有不同的权限

1. **普通用户和企业用户**可以查看自己的用户信息，修改自己的密码等操作。
2. **管理员**可以进行角色设置以及角色和权限的绑定。
3. **创世神**拥有创造权限，创建消费者/物流公司/供应商以及创建订单等。（直接使用原网页）
